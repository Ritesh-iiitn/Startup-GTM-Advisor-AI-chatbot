# LaunchPilot AI – Startup GTM Advisor

LaunchPilot AI is a purpose-built AI agent designed to help early-stage startups plan and execute their go-to-market strategy.  
It provides structured, execution-focused guidance on growth, outreach, ICP definition, and operational automation.

This project was built as part of a software engineering assignment for Thinkly Labs, focusing on designing a product-like AI experience rather than a generic chatbot.

## 🎯 Problem Statement

Early-stage founders often struggle with:

- Designing effective go-to-market strategies
- Identifying ideal customer profiles
- Planning outreach and growth experiments
- Structuring operational workflows

LaunchPilot AI aims to provide practical, structured guidance to reduce this ambiguity.

---

## 🧠 Key Features

- Purpose-built AI GTM advisor (not a general chatbot)
- Structured strategic responses with execution steps
- Suggested prompt onboarding experience
- Clean SaaS-style UI design
- Responsive layout for different screen sizes
- Real-time conversational interface

---

## 🏗 Tech Stack

- Next.js (App Router)
- React + Tailwind CSS
- AI SDK integration
- Groq / LLM API
- Vercel deployment

---

## 💡 Why This Topic

Thinkly Labs builds AI agents focused on operational efficiency and go-to-market automation.  
This project explores a similar domain by creating an AI advisor that helps startups structure growth strategy and execution workflows.

The goal was to demonstrate product thinking, UX clarity, and practical AI integration rather than model complexity.
